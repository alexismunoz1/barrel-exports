# barrel-exports

Analyzes and migrates barrel export patterns in TypeScript/React projects to reduce bundle size. Detects barrel files (index.ts re-exports), checks for side effects, measures bundle impact, and safely rewrites imports to direct paths. Proven to reduce page chunks by up to 19x in Next.js apps.

## Installation

```bash
npx skills add alexismunoz1/barrel-exports
```
